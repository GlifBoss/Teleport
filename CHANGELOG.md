# Изменения в программе

## Телепорт 1.4

* Первая стабильная версия программы.

## Телепорт 1.5

* Добавлено переключение темы интерфейса в настройках, 
  в том числе добавлен режим Auto для синхронизации с настройкой системы.
* Добавлен вариант программы с английским языком интерфейса.
* Системный скролл бар заменён на кастомный.
* Добавлен прогресс бар и цифровой индикатор выполнения.
* Ширина окна увеличивается пропорционально размеру шрифта.
* Косметическая доработка интерфейса.
* Оптимизация кода.

# Changelog

## Teleport 1.4

* The first stable version of the program.

## Teleport 1.5

* Added interface theme switching in settings, 
  including an "Auto" mode to sync with system settings.
* Added an English language interface version of the program.
* Replaced the system scroll bar with a custom one.
* Added progress bar and digital progress indicator.
* The window width increases proportionally to the font size.
* Cosmetic interface improvements.
* Code optimization.
